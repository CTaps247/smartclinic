# User Stories

## Admin

### Manage Doctors
As an Admin, I want to add, update, and remove doctors so that I can manage doctor records.

**Acceptance Criteria:**
- Add doctor
- Update doctor
- Delete doctor

**Priority:** High  
**Story Points:** 8  

---

### Manage Users
As an Admin, I want to manage patients and doctors so that I can control system access.

**Acceptance Criteria:**
- View users
- Disable accounts

**Priority:** High  
**Story Points:** 5  

---

## Doctor

### View Appointments
As a Doctor, I want to view my scheduled appointments so I can manage my workload.

**Priority:** High  
**Story Points:** 5  

---

### Manage Consultations
As a Doctor, I want to update consultation details so patient records are accurate.

**Priority:** High  
**Story Points:** 8  

---

## Patient

### Register/Login
As a Patient, I want to register and log in so I can use the system.

**Priority:** High  
**Story Points:** 3  

---

### Book Appointments
As a Patient, I want to book appointments so I can schedule visits.

**Priority:** High  
**Story Points:** 5
