# User Stories

## Admin

### Manage Doctors

As an Admin, I want to manage doctor records so that the system stays up to date.

**Acceptance Criteria:**

* Add new doctors
* Update doctor details
* Delete doctor records

**Priority:** High
**Story Points:** 8

---

### Manage Users

As an Admin, I want to manage patient and doctor accounts so that system access is controlled.

**Acceptance Criteria:**

* View all users
* Disable or enable accounts
* Ensure only active users can log in

**Priority:** High
**Story Points:** 5

---

### Manage Specializations

As an Admin, I want to manage doctor specializations so patients can search correctly.

**Acceptance Criteria:**

* Add specialization categories
* Update existing specializations
* Remove invalid entries

**Priority:** Medium
**Story Points:** 3

---

### View System Reports

As an Admin, I want to view system usage reports so I can monitor platform activity.

**Acceptance Criteria:**

* View appointment counts
* View doctor activity
* View patient activity

**Priority:** Medium
**Story Points:** 5

---

### Monitor Appointments

As an Admin, I want to monitor all appointments so I can ensure system integrity.

**Acceptance Criteria:**

* View all appointments
* Filter by doctor or patient
* Detect scheduling conflicts

**Priority:** Medium
**Story Points:** 5

---
